# 申城学联 - 设计规范文档

## 1. 设计理念

**简洁现代，青春活力**
- 面向高校学生群体的年轻化设计
- 清晰的信息层级与直观的操作流程
- 轻量级交互，降低学习成本

---

## 2. 色彩系统

### 2.1 主色调（浅蓝色系）

| 色彩名称 | 色值 | 使用场景 | 说明 |
|---------|------|---------|------|
| 主色 - Primary | `#4A90D9` | 导航栏、按钮、链接、高亮 | 品牌核心色，代表智慧与信任 |
| 主色浅 - Primary Light | `#7BB3E8` | 悬停状态、渐变起点 | 温和的蓝色过渡 |
| 主色浅 - Primary Lighter | `#E3F0FD` | 背景色、卡片底色、标签背景 | 极浅蓝，营造轻松氛围 |

### 2.2 辅助色

| 色彩名称 | 色值 | 使用场景 | 说明 |
|---------|------|---------|------|
| 成功绿 | `#52C41A` | 成功提示、可用状态 | 积极正向 |
| 警告橙 | `#FA8C16` | 警告提示、待处理 | 引起注意但不紧急 |
| 错误红 | `#F5222D` | 错误提示、拒绝操作 | 明确警示 |
| 信息蓝 | `#1890FF` | 信息提示、链接 | 品牌延伸 |

### 2.3 中性色

| 色彩名称 | 色值 | 使用场景 | 说明 |
|---------|------|---------|------|
| 纯白 | `#FFFFFF` | 主背景、卡片背景 | 干净清爽 |
| 浅灰背景 | `#F5F7FA` | 次级背景、分割区域 | 柔和对比 |
| 边框灰 | `#E0E6ED` | 边框、分割线 | 清晰分隔 |
| 文本主色 | `#2C3E50` | 标题、正文 | 高可读性 |
| 文本次色 | `#5A6C7D` | 描述、辅助信息 | 层级分明 |
| 文本占位 | `#A0AEC0` | 占位符、禁用状态 | 弱化处理 |

### 2.4 功能色标签示例

## 4. 间距系统

采用 **8px 基础网格系统**，所有间距为 8 的倍数。

| 间距名称 | 数值 | 使用场景 |
|---------|-----|---------|
| XS | 4px | 紧密排列元素、图标与文字间距 |
| SM | 8px | 相关元素间距、标签内边距 |
| MD | 16px | 卡片内边距、列表项间距 |
| LG | 24px | 区块间距、表单组间距 |
| XL | 32px | 页面区块分隔 |
| XXL | 48px | 大区块分隔、页面上下边距 |

### 4.1 组件内边距规范

- **按钮**：`8px 16px`（小）、`12px 24px`（中）、`16px 32px`（大）
- **输入框**：`8px 12px`
- **卡片**：`16px`
- **标签/徽章**：`4px 8px`

## 8. 按钮系统

## 15. 动画与过渡

### 15.1 过渡时长

- **快速**：150ms - 悬停、小元素
- **标准**：250ms - 按钮、卡片
- **慢速**：350ms - 页面切换、弹窗

### 15.2 缓动函数

- **标准**：`cubic-bezier(0.4, 0, 0.2, 1)`
- **进入**：`cubic-bezier(0, 0, 0.2, 1)`
- **离开**：`cubic-bezier(0.4, 0, 1, 1)`

### 15.3 常用动画

- **淡入淡出**：opacity 0 ↔ 1
- **上滑出现**：translateY(20px) ↔ translateY(0)
- **缩放**：scale(0.95) ↔ scale(1)
- **旋转**：rotate(0deg) ↔ rotate(180deg)

### 16.1 浮动按钮

- **距离**：底部 24px，左侧 24px

- **位置**：左下角，浮动按钮上方

## 17. 可访问性

### 17.1 颜色对比度

- 正文文字：对比度 ≥ 4.5:1
- 大文字（18px+）：对比度 ≥ 3:1
- 图标与组件：对比度 ≥ 3:1

### 17.2 键盘导航

- 所有交互元素支持 Tab 键访问
- 焦点可见：2px 蓝色边框 `#4A90D9`

### 17.3 屏幕阅读器

- 图片包含 alt 描述
- 表单元素关联 label
- 语义化 HTML 标签
