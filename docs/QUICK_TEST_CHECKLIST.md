## 快速故障排查

| 问题 | 可能原因 | 解决方案 |
|-----|---------|---------|
| 后端启动失败 | 端口被占用 | 修改端口或关闭占用进程 |
| 前端无法连接后端 | CORS配置错误 | 检查后端CORS设置 |
| 数据库错误 | 数据库未初始化 | 运行 npm run init-db |
| 登录失败 | 密码加密问题 | 检查bcrypt配置 |
| 图片上传失败 | uploads目录不存在 | 创建uploads目录 |
